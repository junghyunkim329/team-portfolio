선박 빅데이터 공모전

25.11.25 ~ 25.12.04

Project Overview

  • 해상 빅데이터 기반 선박 사이버 리스크 실험 디지털 트윈 프레임워크

Motivation

  • 디지털 운항 장비 활용도 증가로 인한 사이버 리스크 증가
  • 구형 프로토콜의 취약한 통신 구조로 인한 신호 위/변조 위험
  • 실시간 이상 감지 체계 부족

Activities

  • 실제 선박의 실시간 연동형 시뮬레이션을 사용하여, 대상 선박과 시뮬레이션 모델을 양방향으로 연결하여 공격 시나리오 실험에 대한 분석을 수행 아이디어 제안

Members

  • 김정현(PM)
  • 양▭권(PA)
  • 조▭기(PA)

Expected Effects

  • 모의 공격이 어려운 실 선박 대비 자유로운 사이버 리스크 관련 테스트 가능
  • 공격 실험 결과를 바탕으로 실제 선박이 공격을 탐지하고 대응 할 수 있는 기준을 정량화
  • 차세대 연구자들에게 기반 데이터를 제공함으로써 분야에 지속적인 기여 가능
